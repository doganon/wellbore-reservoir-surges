# wellbore-reservoir-surges
Modeling and analysis of valve-induced pressure surges in petroleum wells using a coupled wellbore–reservoir approach for rapid reservoir diagnostics.

This repository implements a coupled wellbore–reservoir modeling framework for analyzing pressure transients generated by rapid valve actions (water hammer). The model integrates transient pipe-flow equations with analytical and numerical porous-media flow formulations to simulate pressure-wave propagation and its interaction with the reservoir. The approach enables short-duration, high-frequency diagnostics of near-wellbore properties such as permeability, porosity, and fracture characteristics.
